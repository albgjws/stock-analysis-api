# 股票分析工具

一个基于 Node.js 的全栈股票分析工具，输入股票代码或名称即可获取技术分析、趋势预测和买卖信号。

## 功能

- 🔍 **股票搜索** — 支持股票代码、名称、拼音模糊搜索
- 📊 **K线图表** — 交互式K线图，叠加MA均线、布林带
- 📈 **技术指标** — MACD / RSI / KDJ 指标图表
- 🔮 **趋势预测** — ARIMA 时间序列模型预测未来走势
- 💡 **买卖信号** — 多指标加权评分系统，提供操作建议

## 技术栈

| 层 | 技术 |
|---|---|
| 后端 | Node.js + Express + TypeScript |
| 前端 | React + Vite + Ant Design + ECharts |
| 数据 | stock-sdk（腾讯/东方财富免费接口） |
| 预测 | ARIMA（WebAssembly 加速） |
| 缓存 | node-cache + 文件系统二级缓存 |

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式（同时启动前端和后端）
npm run dev

# 访问 http://localhost:5174
```

## 生产部署

```bash
# 构建前端
npm run build

# 启动服务（Express 提供前端静态文件 + API）
npm start

# 访问 http://localhost:3001
```

## API 接口

| 端点 | 说明 |
|---|---|
| `GET /api/stock/search?q=茅台` | 搜索股票 |
| `GET /api/stock/:code/analysis` | 完整分析（含预测+信号） |
| `GET /api/health` | 健康检查 |

## 使用示例

1. 打开浏览器访问首页
2. 搜索栏输入"600519"或"茅台"
3. 选择"贵州茅台"进入分析页面
4. 查看K线图、技术指标、趋势预测和买卖信号

## 买卖信号说明

信号基于 RSI、MACD、布林带、均线排列、KDJ、成交量等指标加权评分：

| 总分 | 信号 |
|---|---|
| ≥ 5 | 强烈买入 🟢 |
| 2 ~ 4 | 买入 🟢 |
| -1 ~ 1 | 持有 ⚪ |
| -4 ~ -2 | 卖出 🔴 |
| ≤ -5 | 强烈卖出 🔴 |

## 数据说明

- 数据来源：腾讯财经 + 东方财富（免费接口）
- 数据延迟：数秒至数分钟
- 首次加载后自动缓存，可离线查看
